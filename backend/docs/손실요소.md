위치에 따라 변하는 요소들
거리 계산 (R)
   R = calc_distance_to_target(target_positions, satellite_position)
타겟 위치와 위성 위치에 따라 거리가 달라짐
거리 = ||타겟 위치 - 위성 위치||
시간 지연 (td)
   td = calc_time_delay(R_2way)  # R_2way = 2 × R
거리에 비례: td = 2 × R / c
거리가 멀수록 Echo 지연 시간이 길어짐
안테나 게인 (ant_gain)
   ant_gain = calc_antenna_gain_simple(       target_positions, satellite_position, beam_direction, ...   )
타겟 방향에 따라 안테나 게인이 달라짐
빔 중심에서 멀어질수록 게인 감소
대기 손실 (atmospheric_loss)
   atmospheric_loss = calc_atmospheric_loss(beam_direction, satellite_position)
빔 방향과 위성 위치에 따라 달라짐
지구 표면에 가까울수록 손실 증가
Echo 신호 세기 (c)
   c = sqrt(c1 * σ / R⁴)
거리의 제곱에 반비례: Pr ∝ 1/R²
거리가 멀수록 Echo 신호가 약해짐
